# Rocket Music Hub

Plataforma interativa para comunidade e hub de música.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias de ponta:

* **Vite** - Build tool ultra-rápida
* **TypeScript** - Tipagem estática para JavaScript
* **React** - Biblioteca para construção de interfaces
* **shadcn-ui** - Componentes de interface acessíveis e customizáveis
* **Tailwind CSS** - Framework CSS utilitário

## ⚙️ Pré-requisitos

Certifique-se de ter o [Node.js](https://nodejs.org/) (versão 18 ou superior) instalado em sua máquina. Recomenda-se o uso do `npm` como gerenciador de pacotes.
